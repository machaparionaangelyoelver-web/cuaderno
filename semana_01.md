## 📖 Semana 01: Fundamentos de la Tecnología Web

En esta primera semana exploramos cómo funciona la **World Wide Web**, los estándares que la sustentan, los roles de **frontend y backend**, y herramientas claves para comenzar a desarrollar.

---

### 🌐 Conceptos Clave

| Concepto | Descripción | Imagen |
|----------|-------------|--------|
| **Frontend** | Interfaz que ve el usuario (HTML, CSS, JS). Se enfoca en diseño y experiencia. | <img src="https://cdn-icons-png.flaticon.com/512/919/919827.png" width="60"/> |
| **Backend** | Lógica, seguridad y bases de datos en el servidor. Procesa peticiones del cliente. | <img src="https://cdn-icons-png.flaticon.com/512/5968/5968322.png" width="60"/> |
| **HTTP / HTTPS** | Protocolos que permiten la comunicación entre navegador y servidor. HTTPS agrega cifrado. | <img src="https://upload.wikimedia.org/wikipedia/commons/5/5b/HTTP_logo.svg" width="80"/> |
| **DNS** | Convierte nombres como `google.com` en direcciones IP. | <img src="https://upload.wikimedia.org/wikipedia/commons/0/0e/DNS_Server_icon.png" width="60"/> |
| **Open Web Platform** | Ecosistema abierto de tecnologías (HTML5, CSS3, JS, APIs). | <img src="https://cdn-icons-png.flaticon.com/512/732/732190.png" width="60"/> |

### 🧪 Laboratorio 01: Git Bash + Visual Studio Code

| Herramienta | Uso clave | Comandos / Funciones |
|-------------|-----------|-----------------------|
| **Git** | Control de versiones y trabajo colaborativo. | `git init`, `git add .`, `git commit -m "mensaje"`, `git push origin main` |
| **GitHub** | Plataforma para alojar repositorios y compartir código. | Crear repos, conectar con `git remote add origin` |
| **VSCode** | Editor de código moderno y personalizable. | Paleta de comandos (Ctrl + Shift + P), extensiones, snippets |
| **Emmet** | Atajos para escribir HTML/CSS rápido. | Ejemplo: `ul>li*5` → lista con 5 elementos |

---

### ⚡ Línea del Tiempo de la Web

| Año | Hito |
|-----|------|
| **1989** | Tim Berners-Lee propone la **World Wide Web** |
| **1991** | Primer sitio web público |
| **1993** | Aparece **Mosaic**, primer navegador gráfico |
| **1995** | Nacen **JavaScript** y el modelo cliente-servidor moderno |
| **2008** | HTML5 se empieza a estandarizar |
| **2015** | Llega **ES6**, clave para JS moderno |
| **2020+** | Frameworks como React, Next.js y Vue dominan el frontend |

---

### 🧭 Comparación Frontend vs Backend

| Aspecto | Frontend | Backend |
|---------|----------|---------|
| Lenguajes | HTML, CSS, JavaScript | Python, Node.js, Java, PHP |
| Función | Presentación visual e interacción | Procesamiento de datos, lógica y APIs |
| Ejemplo | Botón que cambia color al pasar el mouse | Servidor que guarda datos en una base |
| Frameworks | React, Angular, Vue | Django, Express, Spring |
